## The Change Object

```typescript
type Change = {
  start: number,
  end: number,
  value: Value
}
```

--- 

Naive implementation of `change(Change)`:

```
const { start, end, value } = change

const newValue =
  text.slice(0, start)
  .concat(value)
  .concat(text.slice(end, text.length))
```
